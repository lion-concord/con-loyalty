# MVP con-loyalty

## Цель
Собрать и опубликовать Android-приложение программы лояльности КОН как самостоятельный продукт.

## В MVP входит
- вход по номеру телефона
- подтверждение номера по SMS
- профиль пользователя
- баланс баллов КОН
- история операций
- партнёрский модуль «Семь Рек»
- оформление заказа у партнёра
- списание и начисление баллов
- release APK/AAB

## В MVP не входит
- демо-магазин
- моковые покупки
- тестовые витрины без backend
- временные UI-заглушки

## Принципы
- КОН — это баллы лояльности
- сервер является источником истины для баланса и операций
- frontend не хранит критическую бизнес-логику начисления и списания
- ключевые пользовательские сценарии должны работать end-to-end
