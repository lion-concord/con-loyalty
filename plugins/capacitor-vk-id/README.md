# capacitor-vk-id

VK ID authentication plugin for Capacitor

## Install

npm install capacitor-vk-id
npx cap sync

## Configuration

### Android

1. Add VK App ID in android/app/src/main/res/values/strings.xml

2. Add activity in android/app/src/main/AndroidManifest.xml
